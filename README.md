Olivier MEHAYE proposes an original primality test based on Lucas sequences, relying on a refined analysis of the associated congruences and on the behavior of certain recurrent sequences modulo an odd integer. The program implements this test in Python and enables efficient identification of prime numbers. To date, no pseudoprime listed in the OEIS databases is known to fool this test, including Fermat base-5 pseudoprimes, which are among the most challenging candidates. This work is part of an exploratory approach in computational number theory, aiming to provide a conceptual and algorithmic alternative to classical probabilistic primality tests, while remaining fully reproducible through an open-source implementation.

Olivier MEHAYE propose un test de primalité original fondé sur les suites de Lucas, reposant sur une analyse fine des congruences associées et sur le comportement de certaines suites récurrentes modulo un entier impair. Le programme implémente ce test en Python et permet d’identifier efficacement les nombres premiers. Aucun pseudo-premier répertorié à ce jour dans les bases OEIS ne "trompent" ce test et notamment les PP Fermat Base 5 qui sont pourtant les candidats les plus sérieux. Ce travail s’inscrit dans une démarche exploratoire en théorie des nombres computationnelle, visant à offrir une alternative conceptuelle et algorithmique aux tests probabilistes classiques, tout en restant entièrement reproductible grâce à une implémentation open source.
